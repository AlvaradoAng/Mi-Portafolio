Proyecto Pokédex

El proyecto consiste en la creación de una Pokédex interactiva que permita a los usuarios filtrar y buscar a un Pokémon, proporcionando información relevante sobre cada uno, como su nombre, tipo e imagen para poder identificarlo.

  • Visual Studio Code: Como entorno de desarrollo integrado (IDE) para escribir y organizar el código.
