Proyectos:

Proyecto Pokédex

El proyecto consiste en la creación de una Pokédex interactiva que permita a los usuarios filtrar y buscar a un Pokémon, proporcionando información relevante sobre cada uno, como su nombre, descripción, tipo e imagen para poder identificarlo.

Tecnologías Utilizadas:

  • HTML: Para la estructura del sitio web.
  • CSS: Para el diseño y la presentación visual.
  • Angular: Para la creación de una aplicación web dinámica y reactiva.
  • Visual Studio Code: Como entorno de desarrollo integrado (IDE) para escribir y organizar el código.
